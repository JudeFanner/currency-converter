# currency converter

a simple Java app for converting currencies using live exchange rates

## setup

1. clone this repository
2. obtain an API key from [exchangerate-api](https://www.exchangerate-api.com/)
3. run the application and enter your API key when prompted

## note

this project requires a valid ExchangeRate-API key to function. for security reasons, the API key is not included in this repository. users must provide their own key.

## features

- up to date currency conversion
- favorite currency list
- default currency setting
- manual ate refresh functionality
